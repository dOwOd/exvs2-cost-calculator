# ゲーム戦術知識

EXVS2 における戦術判断の知識をまとめたドキュメント。
単純なルールでは表現できない複雑なトレードオフを理解するために使用する。

## コストと機体性能の関係

高コストの機体ほど性能が高く、低コストほど性能が低い。

| コスト | 性能 | 特徴 |
|--------|------|------|
| 3000 | 最高 | 攻撃力・機動力が高く、派手な武装が多い。ゲームの花形 |
| 2500 | 高い | コスト相応の高性能 |
| 2000 | 低い | 攻撃力・機動力が低い |
| 1500 | 最低 | 性能は最も低いがコスト効率が良い |

## コストオーバーのペナルティ

### 高コストほどペナルティが深刻

同じ残コストでも、高コスト機のほうが復帰耐久が大幅に低くなる。

**同じ残コスト500での復帰耐久率（初期耐久680の場合）:**

| コスト | 復帰耐久 | 復帰率 |
|--------|----------|--------|
| 3000 | 120 | 17.6% |
| 2500 | 140 | 20.6% |
| 2000 | 170 | 25.0% |

→ 3000コストはコストオーバー時のデメリットが非常に大きい

### 具体例: 3000 + 2000（両機とも耐久680）

**パターンA: 3000先落ち → 2000撃墜**
- 残コスト: 6000 → 3000 → 1000
- 2000の復帰耐久: **340**（50%）

**パターンB: 2000先落ち → 3000撃墜**
- 残コスト: 6000 → 4000 → 1000
- 3000の復帰耐久: **230**（約34%）

→ 同じ残コスト1000でも、3000コストのほうが復帰耐久が110低い

### 具体例: 3000 + 2500（両機とも耐久680）

**パターンA: 3000先落ち → 2500撃墜**
- 残コスト: 6000 → 3000 → 500
- 2500の復帰耐久: **140**

**パターンB: 2500先落ち → 3000撃墜**
- 残コスト: 6000 → 3500 → 500
- 3000の復帰耐久: **120**

→ 同じ残コスト500でも、3000のほうが20低い

### 具体例: 2500 + 2000（両機とも耐久680）

**パターンA: 2500先落ち → 2000撃墜**
- 残コスト: 6000 → 3500 → 1500
- 2000の復帰耐久: **510**（75%）

**パターンB: 2000先落ち → 2500撃墜**
- 残コスト: 6000 → 4000 → 1500
- 2500の復帰耐久: **410**（60%）

→ 同じ残コスト1500でも、2500のほうが100低い

## 高コストが低耐久で復帰すると何が起きるか

1. **高コスト機が自由に動けなくなる**
   - 耐久が低いため慎重な立ち回りを強いられる
   - 性能の高い機体を活かせない = コストの無駄

2. **2vs1の構図になる**
   - 相手は性能の低い低コスト機を無視
   - 耐久の低い高コスト機を2機で集中攻撃
   - 結果として負ける

## セオリー: 高コスト先落ち

上記の理由から、**高コストが先に撃墜され、次に低コストが撃墜される**のがセオリー。

### セオリーのリスク

高コスト先落ちにもリスクがある：

- 高コストが先に落ちた後、立ち回り次第では**続けて高コストが撃墜される**可能性
- その場合、**EXオーバーリミットが発動できずに負ける**

→ これがこのゲームの奥深い点

### セオリーの適用範囲

| 編成タイプ | セオリー適用 | 備考 |
|------------|--------------|------|
| 3000 + 低コスト | **強く適用** | 3000のコストオーバーペナルティが顕著 |
| 2500 + 低コスト | 適用 | 高コスト先落ちが基本 |
| 低コスト同士 | **プレイヤー次第** | EX発動タイミングと耐久管理のトレードオフ |

## EXオーバーリミットの仕様

### 発動条件

残コスト ≤ min(A機コスト, B機コスト)

### 効果

| 効果 | 詳細 |
|------|------|
| 発動時間 | 10秒間 |
| 攻撃力 | 上昇 |
| 機動力 | 上昇 |
| バリア | 敵の攻撃を100ダメージ分防ぐ |
| 耐久回復 | 発動中に与えたダメージの30%を自身の耐久値として回復 |

### 発動タイミングの注意

- 僚機が撃墜されてから復活するまで**約3秒**かかる
- その間はEXオーバーリミットが発動しない
- 両機が瀕死の状態で片方が撃墜されると、残りの機体も3秒以内に撃墜されてEX発動前に負ける危険性がある

## 低コスト同士の編成の特徴

### コストオーバーが発生しない編成

以下の編成では、撃墜されてもフル耐久で復帰できる：

- 2000 + 2000
- 1500 + 1500

### フル耐久復帰のデメリット

EXオーバーリミット発動時にフル耐久だと、**耐久回復の恩恵を受けにくい**。

- 耐久回復は「与ダメージの30%」だが、回復上限はフル耐久まで
- フル耐久で発動すると回復量が無駄になる

### 強気な立ち回りの選択

そのため、僚機が撃墜されてEXが発動する前に**ある程度耐久値を減らしておく強気な立ち回り**を行うことがある。

### 強気な立ち回りのリスク

耐久値を減らしすぎると危険。

**具体例: 2000 + 2000、残コスト4000の場面**

1. 両機とも瀕死の耐久値
2. 片方の機体が撃墜される
3. EX発動まで約3秒かかる
4. その間にもう片方も撃墜される
5. **EXオーバーリミットが発動できずに負ける**

## 編成別の指針

### 3000 + 3000

- **特徴**: 最大火力だが、コスト効率は最も悪い
- **基本方針**: 両者ともに落ちないように立ち回る。1落ちで即EX発動可能
- **トレードオフ**: 火力 vs 継戦能力
- **注意点**: 2落ちで敗北のため、リスク管理が最重要

### 3000 + 2500

- **特徴**: 高火力と安定性のバランス型
- **基本方針**: 3000が前衛、2500がサポートしつつ自衛。3000先落ちがセオリー
- **トレードオフ**: 3000先落ちでEX早期発動 vs 2500先落ちで3000温存
- **注意点**: 2500が先に2回落ちると3000がコストオーバーで出撃

### 3000 + 2000

- **特徴**: 3000を活かしつつ、2000が複数回出撃可能
- **基本方針**: 3000が主軸、2000は生存重視でサポート。3000先落ちがセオリー
- **トレードオフ**: 2000の落ち数管理が重要
- **注意点**: 2000が先に落ちると3000のコストオーバーが深刻（復帰耐久230）

### 3000 + 1500

- **特徴**: 3000が2回出撃可能な編成
- **基本方針**: 3000が積極的に前に出る。1500は援護に徹する
- **トレードオフ**: 3000の2回出撃 vs 1500の脆さ
- **注意点**: 1500が落とされると一気に不利になる

### 2500 + 2500

- **特徴**: バランス型。どちらも2回出撃可能
- **基本方針**: 両者で連携して戦う。片方に負担を集中させない
- **トレードオフ**: 安定性 vs 爆発力の欠如
- **注意点**: 3000編成に火力負けしないよう、ダメージレースを意識

### 2500 + 2000

- **特徴**: 安定した編成。両者とも複数回出撃可能
- **基本方針**: 2500が前衛寄り、2000がサポート。2500先落ちがセオリー
- **トレードオフ**: 継戦能力 vs 火力
- **注意点**: 2000が先に落ちると2500のコストオーバーが発生

### 2500 + 1500

- **特徴**: 2500を活かしつつ、コスト効率を追求
- **基本方針**: 2500が主軸。1500は援護と囮
- **トレードオフ**: コスト効率 vs 1500の脆さ
- **注意点**: 1500が連続で落とされると厳しい

### 2000 + 2000

- **特徴**: 最も安定した編成。両者3回出撃可能。コストオーバーなし
- **基本方針**: 両者で粘り強く戦う
- **トレードオフ**: 安定性 vs 火力不足
- **EX戦術**: フル耐久復帰のため、EX発動前に適度に耐久を減らす立ち回りも選択肢
- **注意点**: ダメージレースで不利になりやすい。両機瀕死でのEX発動失敗に注意

### 2000 + 1500

- **特徴**: コスト効率重視の編成
- **基本方針**: 両者で連携しつつ、落ち数を管理
- **トレードオフ**: 出撃回数 vs 1機あたりの性能
- **注意点**: 個々の機体性能で劣るため、連携が必須

### 1500 + 1500

- **特徴**: 最大出撃回数だが、火力・耐久ともに低い。コストオーバーなし
- **基本方針**: 数で押す。1機あたりの仕事量を意識
- **トレードオフ**: 出撃回数 vs 性能差
- **EX戦術**: フル耐久復帰のため、EX発動前に適度に耐久を減らす立ち回りも選択肢
- **注意点**: 高コスト機に一方的にやられないよう注意。両機瀕死でのEX発動失敗に注意

## パターン評価の観点

パターンを評価する際は、以下の観点を総合的に判断する：

1. **総耐久値**: 高いほど継戦能力がある
2. **EX発動可能性**: 発動条件を満たせるか
3. **EX発動タイミング**: 早すぎず遅すぎないか
4. **コストオーバーの有無と量**: 許容範囲内か
5. **耐久の分布**: 片方に偏りすぎていないか
6. **最短敗北耐久**: 最悪ケースでの耐久値
7. **耐久回復の余地**: EX発動時に回復できる余地があるか

## 注意事項

- 上記はあくまで一般的な指針であり、機体の性能や相性、プレイヤースキルによって最適解は変わる
- 実際のゲームでは状況判断が重要であり、セオリー通りにいかないことも多い
- このドキュメントは随時更新される可能性がある
