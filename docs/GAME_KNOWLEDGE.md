# ゲーム戦術知識

EXVS2 における戦術判断の知識をまとめたドキュメント。
単純なルールでは表現できない複雑なトレードオフを理解するために使用する。

## コストと機体性能の関係

高コストの機体ほど性能が高く、低コストほど性能が低い。
ただし、各コストには固有の役割と立ち位置があり、単純な上下関係ではない。

> **注意**: 以下はあくまで大まかな指標。武装特徴や機動性、時限強化の有無などは同じコストでもまちまちなため、すべての機体がここに書かれている立ち位置・役割で正しく言い表されるわけではない。

| コスト | 性能 | 特徴 |
|--------|------|------|
| 3000 | 最高 | 攻撃力・機動力が高く、派手な武装が多い。ゲームの花形 |
| 2500 | 高い | バランスが良く、どのコストと組んでも安定 |
| 2000 | 低い | 性能控えめだが総耐久値が高く打たれ強い |
| 1500 | 最低 | 特化した一芸で性能差を覆すジャイアントキリングのコスト帯 |

### 3000コスト

最強クラスの主人公後期搭乗機やラスボス機が多く属するコスト帯。

**強み:**
- 非常に高性能で、特にブースト性能が高い（平均BD入力可能数: 7回、一部は8回）
- 他コストと比べて行動できる猶予が数段上で、単体性能では他コストに対して確実に有利

**弱み・リスク:**
- 最大コスト故に撃墜時の被害が非常に大きく、2回撃墜でコストを使い果たし敗北
- 1対1の状況では必ずダメージ勝ちしなければ、負けの展開に一気に傾く
- 今作では1回落ちただけで「味方が落ちてもゲームは終わらない（EXオーバーリミットが解放される）が、自分は落ちたら即敗北」という集中放火されやすい状況が生まれやすい

**EXVS2での先落ちリスク:**
- 従来作品では3000が先に落ちるのがセオリーだったが、今作ではEXオーバーリミットの仕組みにより「3000の先落ちはリスク」と考えるプレイヤーもいる

**役割:**
- 実戦では前衛を担当するのが基本
- 相手に接近した上でダメージを受けない確かな技量が求められる
- 相方には2000か2500コストを選んでもらい、フォローしあえる距離を維持しながら敵チームに圧をかける

### 2500コスト

初心者にもおすすめできる、バランスの良いコスト帯。

**強み:**
- まずまず高めの機体性能に加えて、相性が特別悪いコストが存在しない
- どのコストと組んでも恰好がつく安定性の高さが魅力
- 幅広い戦略であらゆる戦局に柔軟に対応できる

**弱み:**
- 1試合中の総耐久力が低め
- ミスが重なる初心者帯では不安要素がある

**役割:**
- 3000と組むと後衛
- それ以外と組むとダブル前衛
- 機体によっては3000と組んでも前衛を担当
- 相方によって役割が大きく変化するのが特徴

### 2000コスト

前期主人公機や脇役ポジションの機体が属することの多い中コスト帯。

**強み:**
- 2500と比較して総耐久値が高く、単純に打たれ強い
- 総耐久値の高さから覚醒ゲージの管理がしやすいという大きな利点がある

**弱み:**
- 2500と比較すると機体性能がやや控えめ

### 1500コスト

量産機のリーダー機などが属する最低コスト。

> **重要**: 1500は2500や2000の下位的な存在ではない。低い耐久とブースト性能を引き換えに、特化した一芸を活かして敵機との性能差を覆すジャイアントキリングのコスト帯。

**強み:**
- 不安定ながら一芸に熟達した機体が多い
- 相手が対処を間違えると3000に匹敵する力で試合を壊せる

**弱み:**
- 得意でない分野では2000をさらに下回る性能も多々あり、とにかくピーキー
- 耐久力の低さから覚醒の使いどころが圧倒的に難しい
- ブースト性能もかなり低く、何をするにも高い経験とテクニック、機体個別のやり込みが必要
- 耐久調整に極めて癖があり、低コストの中では3000との相性が悪い
- EXオーバーリミットが発動できない落ち順がほぼ必ず1通り以上存在し、落ち順にも難儀する

**役割:**
- 他コストと組んで前衛をする必要がある
- 抑えられた総合性能で3000と同じ役割（前衛）をこなすコスト
- 機体に求められている役割を果たせないとあっという間に溶かされる
- 初心者にはおすすめできないコスト帯

## コストオーバーのペナルティ

### 高コストほどペナルティが深刻

同じ残コストでも、高コスト機のほうが復帰耐久が大幅に低くなる。

**同じ残コスト500での復帰耐久率（初期耐久680の場合）:**

| コスト | 復帰耐久 | 復帰率 |
|--------|----------|--------|
| 3000 | 120 | 17.6% |
| 2500 | 140 | 20.6% |
| 2000 | 170 | 25.0% |

→ 3000コストはコストオーバー時のデメリットが非常に大きい

### 具体例: 3000 + 2000（両機とも耐久680）

**パターンA: 3000先落ち → 2000撃墜**
- 残コスト: 6000 → 3000 → 1000
- 2000の復帰耐久: **340**（50%）

**パターンB: 2000先落ち → 3000撃墜**
- 残コスト: 6000 → 4000 → 1000
- 3000の復帰耐久: **230**（約34%）

→ 同じ残コスト1000でも、3000コストのほうが復帰耐久が110低い

### 具体例: 3000 + 2500（両機とも耐久680）

**パターンA: 3000先落ち → 2500撃墜**
- 残コスト: 6000 → 3000 → 500
- 2500の復帰耐久: **140**

**パターンB: 2500先落ち → 3000撃墜**
- 残コスト: 6000 → 3500 → 500
- 3000の復帰耐久: **120**

→ 同じ残コスト500でも、3000のほうが20低い

### 具体例: 2500 + 2000（両機とも耐久680）

**パターンA: 2500先落ち → 2000撃墜**
- 残コスト: 6000 → 3500 → 1500
- 2000の復帰耐久: **510**（75%）

**パターンB: 2000先落ち → 2500撃墜**
- 残コスト: 6000 → 4000 → 1500
- 2500の復帰耐久: **410**（60%）

→ 同じ残コスト1500でも、2500のほうが100低い

## 高コストが低耐久で復帰すると何が起きるか

1. **高コスト機が自由に動けなくなる**
   - 耐久が低いため慎重な立ち回りを強いられる
   - 性能の高い機体を活かせない = コストの無駄

2. **2vs1の構図になる**
   - 相手は性能の低い低コスト機を無視
   - 耐久の低い高コスト機を2機で集中攻撃
   - 結果として負ける

## セオリー: 高コスト先落ち

上記の理由から、**高コストが先に撃墜され、次に低コストが撃墜される**のがセオリー。

### セオリーのリスク

高コスト先落ちにもリスクがある：

- 高コストが先に落ちた後、立ち回り次第では**続けて高コストが撃墜される**可能性
- その場合、**EXオーバーリミットが発動できずに負ける**

→ これがこのゲームの奥深い点

### セオリーの適用範囲

| 編成タイプ | セオリー適用 | 備考 |
|------------|--------------|------|
| 3000 + 低コスト | **強く適用** | 3000のコストオーバーペナルティが顕著 |
| 2500 + 低コスト | 適用 | 高コスト先落ちが基本 |
| 低コスト同士 | **プレイヤー次第** | EX発動タイミングと耐久管理のトレードオフ |

## EXオーバーリミットの仕様

### 発動条件

残コスト ≤ min(A機コスト, B機コスト)

### 効果

| 効果 | 詳細 |
|------|------|
| 発動時間 | 10秒間 |
| 攻撃力 | 上昇 |
| 機動力 | 上昇 |
| バリア | 敵の攻撃を100ダメージ分防ぐ |
| 耐久回復 | 発動中に与えたダメージの30%を自身の耐久値として回復 |

### 発動タイミングの注意

- 僚機が撃墜されてから復活するまで**約3秒**かかる
- その間はEXオーバーリミットが発動しない
- 両機が瀕死の状態で片方が撃墜されると、残りの機体も3秒以内に撃墜されてEX発動前に負ける危険性がある

## 覚醒（バースト）システム

EXオーバーリミットとは別のシステム。ダメージを与える・受けることで覚醒ゲージが溜まり、発動すると一時的にパワーアップする。

### ゲージと発動タイミング

- **半覚**: ゲージ50%で発動。効果時間は短いが、1試合で2回以上使える
- **全覚**: ゲージ100%で発動。効果時間が長く強力だが、1試合で使える回数が少ない
- 覚醒中に撃墜される（覚醒落ち）とゲージが大幅に無駄になるため、使いどころが重要
- **抜け覚醒**: 被弾中（コンボ中）に覚醒を発動して脱出する防御的な使い方

### 覚醒回数の重要性

- 基本的に**半覚2回以上**を目指すのが理想
- 覚醒回数が多いほど、覚醒中の攻めや防御の選択肢が増える
- 覚醒ゲージは相方が撃墜された時にも増加する
- 耐久調整が崩れると覚醒回数が減り、大幅に不利になる

### 覚醒タイプ

機体ごとに複数の覚醒タイプから選択可能。代表的なもの:

- **F覚醒（ファイティング）**: 格闘性能が大幅に強化。近接戦向き
- **S覚醒（シューティング）**: 射撃性能が強化。中〜遠距離戦向き

### 覚醒とEXオーバーリミットの違い

| | 覚醒 | EXオーバーリミット |
|---|------|---------------------|
| 発動条件 | ゲージ蓄積（50%/100%） | 残コスト ≤ min(costA, costB) |
| 発動回数 | 1試合で複数回可能 | 1試合で1回のみ |
| チーム/個人 | 各機体が個別に使用 | チーム全体で発動 |
| 効果 | 覚醒タイプに応じた強化 | 攻撃力・機動力上昇、バリア、耐久回復 |

## 低コスト同士の編成の特徴

### コストオーバーが発生しない編成

以下の編成では、撃墜されてもフル耐久で復帰できる：

- 2000 + 2000
- 1500 + 1500

### フル耐久復帰のデメリット

EXオーバーリミット発動時にフル耐久だと、**耐久回復の恩恵を受けにくい**。

- 耐久回復は「与ダメージの30%」だが、回復上限はフル耐久まで
- フル耐久で発動すると回復量が無駄になる

### 強気な立ち回りの選択

そのため、僚機が撃墜されてEXが発動する前に**ある程度耐久値を減らしておく強気な立ち回り**を行うことがある。

### 強気な立ち回りのリスク

耐久値を減らしすぎると危険。

**具体例: 2000 + 2000、残コスト4000の場面**

1. 両機とも瀕死の耐久値
2. 片方の機体が撃墜される
3. EX発動まで約3秒かかる
4. その間にもう片方も撃墜される
5. **EXオーバーリミットが発動できずに負ける**

## 編成別のコスト管理

落ちる順序やタイミングが勝敗を大きく左右する。以下は全コスト組み合わせのセオリーを解説する。

> **注意**: コスト1500は初心者にはおすすめできない。

### 3000 + 3000

シャッフルで頻繁に起きる典型的な事故パターンで、基本的に推奨されない。ただし基本性能の高さと、**EXオーバーリミットが発動できない落ち順パターンがない**点から、いくつかの低コストコンビよりはマシ。

**基本方針:**
- 両機1落ちまたは片方2落ちで敗北するため、前衛後衛に分かれる
- **前衛（先落ち側）**: 被弾覚悟で積極的に暴れる
- **後衛（0落ち側）**: 相方の覚醒が溜まるまでは回避最優先。誰にも見られていない時のみ近付いて援護

**前衛後衛の決め方:**
- 自衛力が高く安定した耐久調整ができる機体が後衛
- 格闘機・自衛力に不安がある機体・時限強化機で生時の性能に難がある機体が前衛
- 万能機同士の場合は武装・対面相性を考慮するか、開幕前に先落ちをアピール

**覚醒の使い方:**
- 前衛は「**相手の覚醒を見てから自分の覚醒を使う**」受動的な使い方が重要
- 相手の覚醒に畳みかけるように覚醒を使い、性能差で押し通す
- 先落ち側は「敵の覚醒は0落ち側に向かって使われる」ことを意識する
- 前衛が安易に攻め覚醒 → 敵の覚醒が0落ち側に飛ぶ → 耐久調整崩壊は典型的な負けパターン

**覚醒回数**: 前衛2回、後衛は全覚1回 or 半覚2回

### 3000 + 2500

シリーズで最もメジャーな組み合わせの一つ。盤面に出せる戦力が最も充実している。

**EX注意点:** 2500が一度も撃墜されずに3000が2回撃墜されると、**EXオーバーリミットが使えないまま敗北**する。

**基本方針:**
- 3000が前衛、2500が後衛
- コストオーバーの影響が大きく、コスオバを受ける側は回避を優先
- コスオバ復帰時は射撃ワンコンボでも負けうるので慎重に立ち回る

**覚醒管理の目安:**
- 2500後落ち: 1回目の覚醒終了時に耐久220以上残せば半覚2回がほぼ狙える
- 3000後落ち: 3000側が耐久200以上残した状態が半覚2回の目安
- 後衛2500は2回覚醒が望ましいが、耐久調整が危うければ全覚1回のほうが安定

**覚醒回数**: 前衛2回、後衛は全覚1回 or 半覚2回

### 3000 + 2000

こちらもメジャーな組み合わせ。コストオーバーの影響が比較的少なく、高コストと組んでも先落ちしにくい。爆発力と安定性のバランスが良い。

**EX注意点:** 2000が一度も撃墜されずに3000が2回撃墜されると、**EXオーバーリミットが使えないまま敗北**する。

**基本方針:**
- 3000が前衛で暴れ、2000は後方から援護
- 3000先落ち前に2000の耐久が減っていると、3000が落ちた後に2000が狙われやすい → 前後シフトも視野に入れる

**覚醒管理の目安:**
- 2000後落ち: 1回目の覚醒終了時に耐久130以上残せば半覚2回がほぼ狙える
- 3000後落ち: 3000側が耐久200以上残した状態が目安
- 後衛2000は半覚2回が理想だが、耐久に不安があれば全覚1回も選択肢

**覚醒回数**: 両者2回以上が理想。後落ち側は全覚1回でも可

### 3000 + 1500

1500のブースト・耐久力の少なさから、3000のコストオーバーが起こりやすい難易度の高い組み合わせ。特に3000側に広い視野と柔軟な立ち回りが求められる。

**EX注意点:** 以下の落ち順ではEXオーバーリミットが発動できないまま敗北する:
- 1500が一度も撃墜されずに3000が2回撃墜
- 1500が2回撃墜された後に3000が初めて撃墜

**基本方針:**
- 1500は基本的に放置に弱いため、覚醒ゲージが溜まりにくいのも問題
- 固定で組む場合は1500を3落ちさせる爆弾戦法を敢行することが多い

### 2500 + 2500

感覚や総耐久的には3000+2000に似ている。やや難しいが事故組み合わせではない。

**EXの強み:** **どのような落ち順でもEXオーバーリミットが発動する**ため安定感がある。

**基本方針:**
- 先落ち側を決めて前衛後衛に分かれてもよいが、**基本はダブル前衛**
- 食らいすぎた側を先落ちとするやり方で問題ない
- 後落ち側は耐久5分の2（40%）で再出撃。高火力コンボでワンパンの恐れがあるが、EXオーバーリミットのバリアで1回はミスが許される
- 近距離戦が得意な機体、荒らし能力に長けた機体、総合力の高い万能機同士で好まれる

### 2500 + 2000

2500が先に落ちて2000が後に落ちるのが基本。ただし2000先落ちでも影響は少ない。2500後衛機と2000前衛寄り機なら、2000先落ちのほうがお互いの強みを活かしやすいこともある。

**EXの特徴:** **どのような落ち順でもEXオーバーリミットが発動する**が、先にEX発動条件を満たす展開になりやすいのがウィークポイント。

**基本方針:**
- 基本は**両前衛**として立ち回りつつ耐久調整
- 覚醒回数が安定し、両機とも柔軟に動きやすいのが強み
- コスオバの緩さから耐久調整がやりやすく、瞬殺されることはまずない
- 3000も1500も絡まないため機体相性が良くないとパワー不足になりがち

**後衛2000の注意:**
- 3000前衛と組んだ気分で後衛をやっていると2500が溶ける
- 2500より先に体力を減らした場合、覚醒を使い先落ちするアドリブ能力も求められる

**後落ち2500の注意:**
- 1落ち前の覚醒を安易に出し惜しみすると2回覚醒できなくなる
- 覚醒落ちしなければ2回目はほぼ確実に溜まるため、恐れず強気に使って試合を動かす

### 2500 + 1500

今作でかなり厳しくなった組み合わせ。

**2つの落ち順作戦:**
- **2500先落ち**: メリットは2500がコスオバしない点。デメリットは2500の2落ちが狙われやすい
- **1500先落ち**: メリットは1500が半覚1回を先に確保できる点。デメリットは2500のコスオバ（復帰耐久 4/5）が発生し耐久調整の難易度が高い

**EXの厳しさ（今作の大きな問題点）:**
- お互い1回ずつ落ちた後に2500が落ちると、**EXオーバーリミットが発動できずに敗北**
- 特に1500→2500の順で落ちた場合「EXが発動できないのに撃墜されたら即負けの2500がコスオバで復帰」という極めて厳しい状況になる
- 1500が十分なダメージを稼げないと「EXを相手より先に発動する」負けパターンにも陥りやすい

以上の負けパターンを両方とも満たしやすいため、事故レベルの組み合わせと評価されている。

### 2000 + 2000

**EXの特徴:** EXオーバーリミットが発動するため安定感はあるが、先に発動条件を満たす展開になりやすいのがウィークポイント。

**基本方針:**
- 長時間ダブルロックを捌けるまともな前衛が存在しないため苦しい
- **両前衛**をベースに、積極的に攻めて立ち回りと覚醒を活かすことが重要
- 相手が格闘寄りなら両後衛で迎撃する立ち回りもあるが、どちらにせよ**ほぼ同時に落ちるのが理想**

**後落ち側の心構え（重要）:**
- 後落ちになった側は**なるべく早めに前に出て落とされに行く**必要がある
- 「後落ちだから引き撃ちでいい」は典型的な負け筋。先落ち側がダブルロックで圧迫され耐久調整が崩壊する
- 後衛寄りの機体でも相方の盾となる位置取りを意識する
- **順落ち（交互に落ちる）が原則**

**覚醒**: 半覚主体で回数を稼ぐのが望ましい

### 2000 + 1500

1500を活かすもう一つの組み合わせ。

**強み:**
- どちらも1落ちまではコスオバしないため、序盤から2機で圧力をかけられる
- 終盤の耐久に比較的余裕があり、覚醒を揃えやすく回数も多く稼げる
- 1500が一度も落ちずに2000が3落ちするケースでしかEX発動不可にならないため、**ほぼ確実にEXオーバーリミットが発動可能**

**理想の落ち順:**
- 基本は1500の2落ちが鉄板（総耐力が最大になる）
- 理想: 1500 → 2000 → 1500
- 許容: 1500 → 1500 → 2000（疑似3020のような耐久調整）
- 危険: 1500&2000 → 2000（残コスト500で2000がコスオバ復帰、耐久約150。1500もワンパン圏内のことが多い）

**立ち回り:**
- 「2000が1500のやりたいことをどれだけサポートできるか」が鍵
- 1500は両前衛を求めるコスト帯であり、2000側も積極的な攻めが必要
- 基礎性能の開きがあるため、覚醒回数で平時の不利を覆すことが課題
- 半覚主体で覚醒を多く使う意識が必須

### 1500 + 1500

シャッフルでもそうそう起こらない超大事故コンビ。

**EXの特徴:** コスト的には**必ずEXオーバーリミットが発動できる**。

**現実:**
- コスト差を覆してダメージを取る以外に勝ち目がない
- 尖った武装を持つ機体が多く相手にとっても面倒ではあるが、機体性能差・ダメージレースの火力差で負けることがほとんど
- 被ダメが安定して多くなるため相手の覚醒も安定してしまう

**立ち回り:**
- 残コスト1500の時に新品（フル耐久）が2機並ぶよう、序盤で2落ちする側を決めておくのが理想
- ただし低耐久の1500は事故に弱く、適切な耐久管理は非常に難しい

## パターン評価の観点

パターンを評価する際は、以下の観点を総合的に判断する：

1. **総耐久値**: 高いほど継戦能力がある
2. **EX発動可能性**: 発動条件を満たせるか
3. **EX発動タイミング**: 早すぎず遅すぎないか
4. **コストオーバーの有無と量**: 許容範囲内か
5. **耐久の分布**: 片方に偏りすぎていないか
6. **最短敗北耐久**: 最悪ケースでの耐久値
7. **耐久回復の余地**: EX発動時に回復できる余地があるか
