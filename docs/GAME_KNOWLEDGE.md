# ゲーム戦術知識

EXVS2 XB における戦術判断の知識をまとめたドキュメント。
単純なルールでは表現できない複雑なトレードオフを理解するために使用する。

## よくある誤解と実際

### コストオーバー

❌ **誤解**: コストオーバー = 常に悪い
✅ **実際**: 状況による

| 観点 | コストオーバーなし | コストオーバーあり |
|------|-------------------|-------------------|
| 耐久効率 | 最大限活用 | 一部無駄になる |
| 相方の負担 | 比較的均等 | 片方に集中しやすい |
| EX発動タイミング | 遅くなりがち | 早まる可能性がある |
| 試合展開 | 安定しやすい | 波が出やすい |

**許容される場面**:

- 高コスト機がEX発動条件を早期に満たしたい場合
- 相方（低コスト機）が耐久に余裕がある場合
- 試合終盤で勝負を決めに行く場合

**避けるべき場面**:

- 序盤での大きなコストオーバー（リカバリーが困難）
- 低コスト機側がコストオーバーする場合（高コスト機の出撃回数が減る）
- 相方が既にダメージを受けている状況

### EX発動と先落ち順

❌ **誤解**: EX発動可能 = 最良のパターン
✅ **実際**: EX発動にもコストがある

EXオーバーリミットは強力だが、発動条件（残コスト ≤ minCost）を満たすには
チームコストを消費する必要がある。早期発動と耐久温存はトレードオフの関係にある。

**高コスト先落ちのメリット**:

- EX発動条件を早期に満たせる
- 高コスト機の火力・機動力を序盤から活かせる
- 相方（低コスト機）が温存される

**高コスト先落ちのデメリット**:

- チームの主軸が早期に1機減る時間帯が生まれる
- 低コスト機に一時的に負担が集中する
- 高コスト機のリスポーン中に押し込まれるリスク

**低コスト先落ちのメリット**:

- 高コスト機が長時間戦線を維持できる
- チーム全体の火力が安定する

**低コスト先落ちのデメリット**:

- EX発動タイミングが遅れる
- 低コスト機の総出撃回数が増え、負担が大きい

### 総耐久値の評価

❌ **誤解**: 総耐久値が高い = 常に良い
✅ **実際**: 耐久の「質」も重要

総耐久値は重要な指標だが、以下の観点も考慮が必要：

- **EX発動タイミング**: 総耐久が高くてもEX発動が遅いと不利になる場合がある
- **耐久の分布**: 片方の機体に耐久が偏ると、その機体が集中攻撃を受けやすい
- **最短敗北耐久**: 一方的に攻められた場合の最低ラインも確認すべき

## 編成別の指針

### 3000 + 3000

- **特徴**: 最大火力だが、コスト効率は最も悪い
- **基本方針**: 両者ともに落ちないように立ち回る。1落ちで即EX発動可能
- **トレードオフ**: 火力 vs 継戦能力
- **注意点**: 2落ちで敗北のため、リスク管理が最重要

### 3000 + 2500

- **特徴**: 高火力と安定性のバランス型
- **基本方針**: 3000が前衛、2500がサポートしつつ自衛
- **トレードオフ**: 3000先落ちでEX早期発動 vs 2500先落ちで3000温存
- **注意点**: 2500が2回落ちると3000は出撃不可

### 3000 + 2000

- **特徴**: 3000を活かしつつ、2000が複数回出撃可能
- **基本方針**: 3000が主軸、2000は生存重視でサポート
- **トレードオフ**: 2000の落ち数管理が重要
- **注意点**: 2000が狙われやすいため、自衛力のある機体が望ましい

### 3000 + 1500

- **特徴**: 3000が2回出撃可能な編成
- **基本方針**: 3000が積極的に前に出る。1500は援護に徹する
- **トレードオフ**: 3000の2回出撃 vs 1500の脆さ
- **注意点**: 1500が落とされると一気に不利になる

### 2500 + 2500

- **特徴**: バランス型。どちらも2回出撃可能
- **基本方針**: 両者で連携して戦う。片方に負担を集中させない
- **トレードオフ**: 安定性 vs 爆発力の欠如
- **注意点**: 3000編成に火力負けしないよう、ダメージレースを意識

### 2500 + 2000

- **特徴**: 安定した編成。両者とも複数回出撃可能
- **基本方針**: 2500が前衛寄り、2000がサポート
- **トレードオフ**: 継戦能力 vs 火力
- **注意点**: 長期戦になりやすいため、耐久管理が重要

### 2500 + 1500

- **特徴**: 2500を活かしつつ、コスト効率を追求
- **基本方針**: 2500が主軸。1500は援護と囮
- **トレードオフ**: コスト効率 vs 1500の脆さ
- **注意点**: 1500が連続で落とされると厳しい

### 2000 + 2000

- **特徴**: 最も安定した編成。両者3回出撃可能
- **基本方針**: 両者で粘り強く戦う
- **トレードオフ**: 安定性 vs 火力不足
- **注意点**: ダメージレースで不利になりやすい

### 2000 + 1500

- **特徴**: コスト効率重視の編成
- **基本方針**: 両者で連携しつつ、落ち数を管理
- **トレードオフ**: 出撃回数 vs 1機あたりの性能
- **注意点**: 個々の機体性能で劣るため、連携が必須

### 1500 + 1500

- **特徴**: 最大出撃回数だが、火力・耐久ともに低い
- **基本方針**: 数で押す。1機あたりの仕事量を意識
- **トレードオフ**: 出撃回数 vs 性能差
- **注意点**: 高コスト機に一方的にやられないよう注意

## パターン評価の観点

パターンを評価する際は、以下の観点を総合的に判断する：

1. **総耐久値**: 高いほど継戦能力がある
2. **EX発動可能性**: 発動条件を満たせるか
3. **EX発動タイミング**: 早すぎず遅すぎないか
4. **コストオーバーの有無と量**: 許容範囲内か
5. **耐久の分布**: 片方に偏りすぎていないか
6. **最短敗北耐久**: 最悪ケースでの耐久値

## 注意事項

- 上記はあくまで一般的な指針であり、機体の性能や相性、プレイヤースキルによって最適解は変わる
- 実際のゲームでは状況判断が重要であり、セオリー通りにいかないことも多い
- このドキュメントは随時更新される可能性がある
