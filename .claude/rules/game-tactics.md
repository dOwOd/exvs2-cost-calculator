# ゲーム戦術知識

このルールはパターン評価やコメント生成に関連する機能の実装時に適用されます。

詳細は `docs/GAME_KNOWLEDGE.md` を参照してください。

## 重要な原則

### コストと性能の関係

- 高コスト = 高性能（攻撃力、機動力、派手な武装）
- 低コスト = 低性能
- 3000コストはゲームの花形

### コストオーバーのペナルティ

- **高コストほどペナルティが深刻**
- 同じ残コストでも、高コスト機のほうが復帰耐久が大幅に低くなる
- 3000コストのコストオーバーは特に深刻

### セオリー: 高コスト先落ち

- 高コストが先に撃墜され、次に低コストが撃墜されるのがセオリー
- 理由: 低コスト先落ちだと高コストがコストオーバーで低耐久復帰 → 2vs1の構図で負ける
- リスク: 高コストが続けて撃墜されるとEX発動できずに負ける

### EXオーバーリミットの仕様

- 発動時間: 10秒
- 効果: 攻撃力・機動力上昇、100ダメージバリア、与ダメージ30%回復
- **発動まで約3秒かかる**（僚機復活待ち）
- 回復上限はフル耐久まで

### 低コスト同士の編成（2000+2000、1500+1500）

- コストオーバーが発生しない → フル耐久復帰
- フル耐久だとEX回復の恩恵を受けにくい
- そのため強気な立ち回りで耐久を減らす選択肢がある
- リスク: 両機瀕死で片方撃墜 → 3秒以内にもう片方も撃墜 → EX発動前に負け

## 単純な良し悪しで判断しない

- **コストオーバー ≠ 常に悪い**: 状況によっては許容される
- **EX発動可能 ≠ 常に最良**: 発動にもコスト（トレードオフ）がある
- **総耐久値が高い ≠ 常に良い**: 耐久の「質」や分布も重要
- **フル耐久復帰 ≠ 常に良い**: EX回復の恩恵を受けにくいデメリットがある

## パターン評価の観点

1. **総耐久値**: 高いほど継戦能力がある
2. **EX発動可能性**: 発動条件を満たせるか
3. **EX発動タイミング**: 早すぎず遅すぎないか
4. **コストオーバーの有無と量**: 許容範囲内か
5. **耐久の分布**: 片方に偏りすぎていないか
6. **最短敗北耐久**: 最悪ケースでの耐久値
7. **耐久回復の余地**: EX発動時に回復できる余地があるか

## コメント生成時の注意

- 断定的な表現（「これが最良」「これは悪い」）は避ける
- トレードオフを説明する
- 編成やプレイスタイルによって最適解が変わることを示唆する
